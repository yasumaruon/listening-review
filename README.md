# listening-review

英語YouTube動画を一文単位で聞いて復習するための、オフライン対応の学習ビューア（PWA）。

自宅のパソコンで取り込んだ動画（音声＋文＋日本語訳）を1ファイルにまとめて書き出し、この端末に読み込んでおくと、**パソコンOFF・ネットなしでも**文単位で再生・復習できます。

## 使い方
1. iPhone/iPad の Safari でこのページ（GitHub Pages の公開URL）を開き、「ホーム画面に追加」。
2. パソコンのアプリで動画を「スマホ用に書き出す」→ OneDrive 経由でこの端末に渡す。
3. アプリの「📥 ファイルから読み込み」で取り込むと、オフラインで復習できる。

## 特徴
- 文単位の再生 / 速度調整（0.5〜2倍）/ 文・全体リピート
- 日本語訳の表示切替、「覚えた」「暗記」の選択
- 音声・学習データは端末内（IndexedDB）にのみ保存。公開の場所には置かない。

個人の学習用ツールです。取り込んだ音声・字幕は第三者に配布しないでください。
